BadStore 2015
=============

Automation scripts for [BadStore2015](https://github.com/otms61/badstore2015)

Requirements
-----

- vagrant
- ansible

Installation
-----

### Vagrant

Download Vagrant from [this site](http://www.vagrantup.com/)

### ansible

~~~ sh
pip install -U ansible 
~~~

Usage
-----

~~~ sh
vagrant up
~~~

and access  [192.168.40.10](http://192.168.40.10/)
